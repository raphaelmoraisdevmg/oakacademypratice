Cores

No css todas as cores são formadas tendo como base três delas vermelho,azul e o verde

podemos usar as cores com seus nomes em inglês, podemos também usar valores hexadecimais para definir uma cor

e tambem e possível definir uma cor com rgb

Hexadecimal:#FFFFFF os dois primeiros digitos mostram o quão vermelho a cor é, os dois proximos o quão verde ela é , e os ultimos dois o quão azul.

RGB:os números que representam cada cor podem ir de 0 a 255, a primeira cor é o vermelho a segunda o verde e a terceira o azul
